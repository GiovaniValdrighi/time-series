Simulation of the distribution of the Dickey Fuller test statistic

TT <-  100
n <- 50
S <- 10000
phi <- 0.8
alpha_f <- function(y){
  ax1 <- 0
  ax2 <- 0 
  for(i in 1:(length(y)-1)){
    ax1 <- ax1 + y[i]*y[i+1]
    ax2 <- ax2 + y[i]*y[i]
  }
  alpha <- ax1/ax2 - 0.2
  return(alpha)
}

s_f <- function(y, alpha){
  axs <- 0
  ax <- 0
  for(i in  1:(length(y)-1)){
    axs <- axs + (y[i+1]-y[i]-alpha*y[i])^2
    ax <- ax + y[i]^2
  }
  axs <- axs/(length(y)-1)
  ax <- ax^0.5
  s <- axs/ax
  return(s)
}

t_f <- function(y){
  alp <- alpha_f(y)
  s <- s_f(y, alp)
  t <- alp/s
  return(t)
}
axi <- 1
all_t <- list()

for(j in 1:S){
  wn <- rnorm(TT+n)
  y <- rep(0, TT+n)
  for(i in 1:(TT+n-1)){
    y[i+1] <- phi*y[i]+wn[i]
  }

  for(i in 1:n){
    all_t[[axi]] <- t_f(y[i:TT+i])
    axi <- axi+1
  }
}
t_num <- unlist(all_t)
hist(t_num)

ARIMA Identification with IPCA data

ipca <- read.csv("ipca.csv")
ipca.ts <- ts(ipca$ipca.acum, start = c(1994, 8), frequency = 12)
plot(ipca.ts, type = "l", main = "IPCA by month", ylab = "IPCA", xlab = "Date")

Steps for the ARIMA model identification:

  • Verify if the series need a transformation to estabilize the variance.
  • Differentiate the series how many times itâs needed to the series ACF decay fast to 0. In this step it can also be used the Dickey Fuller test.
  • Indentify the ARMA(p, q) process by the ACF and PACF plots.

We see that our data show a small variance, and itâs the same along the observed period. Now we differentiate the serie.

diff_series <- function(data, d){
  if(d > 0){
    for(i in 1:d){
      data = diff(data)
    }
  }
  par(mfrow = c(1, 2))
  acf(data, main = "IPCA ACF")
  pacf(data, main = "IPCA PACF")
}

With 2 differentiation the ACF decay fast, but the first lag is negative on the ACF, what is a indicator of over-differenced. So letâs apply the Dickey Fuller test on both models.

Identify ARMA(p, q)

Now letâs look at the IPCA to identify the ARMA(p, q) order.

diff_series(diff1_ipca, d = 0)

As the PACF decays fast, it appears to be a MA model, and the signicant values for the ACF are before the lag 4, so we start with testing the model MA(4).
